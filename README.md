## About Kalkulator Zakat

Aplikasi perhitungan zakat dengan metode fiqih dengan framework laravel.

## Cara Duplikasi

Gunakan perintah berikut di terminal:

```
git clone https://github.com/barengs/zakat.git
cd zakat
composer install
php artisan migrate --seed
php artisan serve
```
